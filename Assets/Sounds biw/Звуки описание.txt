Lvl loops - папка с фоновой зацикленной музыкой уровней. при входе в игру в меню
начинает играть "1 piano" и продолжает играть в течение первого уровня, с увеличением
скорости увеличивается счет игрока, исходя из счета есть 4 уровня с разной музыкой,
 пронумерованной числами соответственно. интервалы счета между уровнями выбираются в процессе 
тестинга сложности, для начала по 15 очков между уровнями.

Buttons - звуки нажатия\отжатия кнопок (игровых и кнопок меню). 
для кнопок меню это рандомный звук из всей папки.
для черной игровой кнопки это звуки типа low, для белой hi. звуки для каждой кнопки 
выбираются случайно внутри типа звуков этой кнопки. то есть для белой это рандомный 
звук из hi, для черной рандомный из low.

gam ovr - звук проигрыша, т.е. неправилно пойманого или пропущенного кубика

catch - звук правильно пойманного кубика
